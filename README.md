用到的函数
=====

```findViewById```让组件和函数对象对应

```getText()```得到对象文本

```Double.valueOf(a)```强制转换 

```
 a类型.valueOf(b)/*把b转换成a类型*/
```

```substring()```

删除字符串最后一位数       

```
M = M.substring(0, M.length() - 1);
```

